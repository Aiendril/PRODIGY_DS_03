# Customer Purchase Prediction using Decision Tree Classifier 

## Project Overview

This project uses a Decision Tree Classifier to predict whether a customer will purchase a product or service based on demographic and behavioral data. The model is trained on a Bank Marketing-style dataset and evaluated using classification metrics.

## Features

- Data preprocessing and encoding
- Train-test data splitting
- Decision Tree model training
- Customer purchase prediction
- Model evaluation using Accuracy Score, Classification Report, and Confusion Matrix

## Technologies Used

- Python
- Pandas
- Scikit-learn
- NumPy
- Matplotlib

## Machine Learning Algorithm

- Decision Tree Classifier
